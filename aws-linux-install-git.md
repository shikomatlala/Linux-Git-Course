
## INSTALL MYSQL 8 COMMUNITY EDITION ON AMAZON LINUX 2023

## 1 Git packages are available using 
```sh
dnf
```
## 2 Install Git
```sh
sudo dnf install git-all
```

## 3 Verify Installation
```sh
git version
```
